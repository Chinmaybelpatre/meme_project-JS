# 🖼️ Meme Generator (JavaScript + API)

This is a simple meme generator built with JavaScript that fetches random memes using an API.

## 🚀 Features
- Loads random memes from an external API.
- Displays meme image, title, and more.
- Easy to use, clean UI.

## 🔗 API Used
- [Meme API](https://meme-api.com/gimme/wholesomememes)

## 💻 Tech Stack
- HTML
- CSS
- JavaScript (Fetch API)

## 📁 How to Use
1. Open `index.html` in your browser.
2. Click the "Generate Meme" button.
3. Enjoy your fresh meme!
